## 프로젝트의 목적
이 프로젝트의 목적은 MVC2 모델을 기반으로 한 게시판 프로젝트입니다.,
사용자가 게시글을 등록, 수정, 삭제, 리스트, 등록 이동 의 기능을 만들었습니다. 
이를 통해 웹 애플리케이션 개발의 기본 원리를 이해하고, 데이터 처리 및 사용자 인터페이스 설계에 대한 연습을 하기 위해 만들었습니다.
